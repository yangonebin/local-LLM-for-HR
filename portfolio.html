Target User

KEY COMPETENCIES 양한빈 · HR AI 플랫폼
05 · COMPETENCIES
이 프로젝트가
보여주는 역량
인사 담당자의 언어로 AI를 기획하고, IT와 HR 사이의 간극을 직접 메운 경험입니다.
IT · HR 브릿지
개발 지식이 없는 인사 담당자도 클릭 한 번으로 AI를 활용할 수 있도록 전체 워크플로우를 설계. IT팀과 HR팀 사이의 언어 장벽을 허물고, 현장에서 바로 쓸 수 있는 도구로 구현.
AI 전문성
GPT 트랜스포머 직접 구현, LoRA 파인튜닝, RAG, RLHF 피드백 루프까지 LLM 전 생애주기를 설계. HR 실무에 필요한 답변이 나오도록 학습 데이터와 파이프라인을 직접 구성.
HR 현업 이해도
노동법·취업규칙·징계절차·연차 규정 등 인사 실무 도메인 지식을 AI 학습 데이터 설계에 직접 반영. 법령 API 13종을 선별하여 실제 HR 업무 질문에 정확한 답변이 나오는 구조로 설계.
보안·컴플라이언스 기획
사내 연봉·인사 데이터의 외부 유출 리스크를 먼저 파악하고 완전 로컬 아키텍처로 해결. 개인정보보호법 준수 근거를 설계 단계에서 확보하여 IT·법무팀 설득이 가능한 구조로 기획.
